# Cyber Heat Radar

Cyber Heat Radar は、サイバーセキュリティ関連ニュースを収集し、話題の「温度感」に基づいて注目トピックを整理する情報収集プロジェクトです。

脆弱性、攻撃観測、ランサムウェア、国内外のセキュリティニュースなどを対象に、日々の把握に役立つ情報を整理します。
